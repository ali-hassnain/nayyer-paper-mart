Frontend for ([Mujmua](https://mujmua.vercel.app/))

## Installing

```bash
# From Source
git clone https://github.com/mosibello/mujmua.git
cd mujmua
```

then install dependencies

```bash
  npm install
```

## Extra options

```bash
# Run a production build, locally
npm run dev

# Serve a production build, locally
npm run build
npm start
```
